# E-Commerce-Website-MERN

## About The Project
This is a full-stack E-Commerce Website built using the MERN stack (MongoDB, Express.js, React.js, Node.js).
Users can register, login, browse products, add items to cart, and place orders.
Admin can manage products and users.

## Technologies Used
- MongoDB
- Express.js
- React.js
- Node.js
- JWT Authentication
- REST API

## Features
- User Registration & Login
- Add to Cart
- Product Management
- Order Management
- Admin Dashboard

## Installation & Setup

### Backend
cd backend  
npm install  
npm start  

### Frontend
cd frontend  
npm install  
npm start
